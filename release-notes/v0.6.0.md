## SpaceCode v0.6.0

本版本新增底部终端面板、电脑控制（cua-driver）、自动重试机制、PPT 办公助手技能包等多项核心功能，并修复了环境面板 Git 变更同步、会话恢复、布局切换等多个问题。

## Highlights

- **底部终端面板**：新增底部终端面板功能，支持切换和拖拽调整高度，提升多任务操作效率
- **电脑控制（cua-driver）**：新增电脑控制功能，集成 cua-driver 后台桌面控制，支持内置安装与进度展示
- **自动重试机制**：新增自动重试功能，支持可恢复错误自动重试并展示进度，增强 LLM 会话稳定性
- **PPT 办公助手**：新增全流程 PPT 办公助手技能包，覆盖从内容规划到样式生成的完整工作流
- **HTML 文件预览**：CodeViewer 新增 HTML 文件预览按钮，InfoPanel 优化 webview 新窗口处理逻辑
- **会话产物汇总修复**：修复恢复会话时产物汇总数据丢失问题，并新增 Git 变更自动同步功能
- **内存优化**：优化聊天与会话模块内存占用，有效防止 OOM（Out of Memory）问题

## Fixes

- 修复恢复会话时产物汇总数据丢失的问题
- 保存会话时保留图片的 contentUrl，避免图片丢失
- 修复会话恢复时的重试残留问题
- 修复切换模式后主面板显示旧会话的问题
- 避免重复注册 webview 的窗口打开处理函数
- 修复查找系统 PATH 中可执行文件的逻辑（Windows 兼容性）
- 修复下载更新时隐藏通知的状态同步问题

## Refactor

- 迁移提示词优化逻辑到前端+主进程，优化架构
- 优化重试机制与消息处理逻辑
- 优化窗口打开处理器的注册逻辑
- 清理代码冗余并简化多处逻辑
