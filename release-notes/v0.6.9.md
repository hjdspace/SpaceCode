## SpaceCode v0.6.9

移动端三大功能上线：本地技能库系统、Clone 进度反馈、Agent Timeline 渲染；同时修复多项 clone 路径与多轮对话 bug，移动端版本同步升级至 v0.2.1。

## Highlights

- **本地技能库系统**：SkillsScreen 改造为 TabBar 布局，新增「技能库」tab；LocalLibrarySource 从 index.json 加载库索引，LocalLibraryInstaller 支持从 asset 一键安装技能到用户目录；LocalLibraryScreen 提供列表/搜索/分类筛选，LocalLibraryDetailPage 展示技能详情；LocalLibraryNotifier 基于 Riverpod 管理状态
- **Clone 进度反馈**：新增 CloneProgress 数据模型与 Stream<CloneProgress> 接口，cloneRepository 改造为流式输出；CloneNotifier 后台 StateNotifier 驱动设置页实时显示 clone 进度与完成状态
- **Agent Timeline 渲染**：新增 TimelineEvent 数据模型与 TimelineAssembler 装配器；ChatMessage 增加 timelineEvents 字段，ChatController 接入装配器，MessageBubble 双路径渲染支持 Agent Timeline 视觉效果
- **GitHub OAuth 调试优化**：硬编码 GitHub OAuth Client ID 作为 debug 构建默认值，便于开发调试，release 构建仍可通过 --dart-define 覆盖
- **桌面端版本号动态读取**：macOS 注册 package_info_plus 插件，配合移动端设置页版本号改为从 package_info_plus 动态读取，避免硬编码

## Fixes

- 修复 clone 路径字符串插值 bug 与 Android 11+ 权限问题
- clone 支持选择系统目录且按仓库名创建子目录，修复目录写入失败和打开目录无反应
- clone 支持自定义路径且保留项目根目录，删除 Timeline 圆点装饰
- 修复多轮对话 assembler 残留导致显示历史内容
- 设置页版本号改为从 package_info_plus 动态读取，避免硬编码 v0.1.0
- 修复本地技能库分类筛选、asset 加载、安装反馈三个 bug
- 修复 MarkdownRenderer 本地图片解析和渲染时序问题
