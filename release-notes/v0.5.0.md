## SpaceCode v0.5.0

新增定时任务管理、全局任务看板、消息队列优先级、Claude-Code Ctrl+S 快捷键，以及桌面应用版本更新方案；修复 cron 解析器、消息队列、会话恢复、中文路径等多项问题。

## Highlights

- **定时任务管理**：新增完整的定时任务管理功能，支持 cron 表达式解析、任务运行日志持久化与过期清理
- **全局任务看板**：实现全局任务看板和任务列表优化，提升多任务管理体验
- **消息队列优先级**：实现消息队列优先级支持（steering vs queue），AI 回复期间支持 Pending Messages
- **Claude-Code Ctrl+S**：实现 Claude-Code Ctrl+S 快捷键功能
- **桌面应用更新**：新增桌面应用版本更新方案与自动状态重置

## Fixes

- 修复 cronParser 三个 bug 并补充测试覆盖
- 修复待发消息发送队列异常时消息丢失问题
- 修正会话恢复和 agent 名称显示问题
- 修复从 JSONL 恢复会话时子代理输出未正确归入 teammateTranscripts 的问题
- 修复中文文件路径乱码及文件名重复显示问题

## Refactor

- 重构聊天输入框的模型和推理模式选择 UI
- 重构智能体选择器为悬浮子菜单形式
- 重构消息发送与暂存逻辑，优化发送流程

## Style

- 替换硬编码样式为 CSS 变量，统一主题适配
