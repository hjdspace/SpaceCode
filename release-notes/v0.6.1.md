## SpaceCode v0.6.1

本版本新增单模型上下文窗口自定义配置、厂商预设地址选择、TodoWrite 工具支持等功能，并修复了会话中止恢复、SCM 文件同步、终端面板等多项问题，同时优化了 Markdown 渲染内存占用。

## Highlights

- **上下文窗口配置**：新增单模型上下文窗口自定义配置，支持灵活调整各模型的上下文长度
- **厂商预设地址**：添加厂商预设地址选择功能，新增多厂商图标与多语言配置，简化模型接入流程
- **TodoWrite 工具**：新增 TodoWrite 工具支持，优化任务管理与 Git diff 展示体验
- **SCM/文件树自动刷新**：在文件工具执行完成后主动刷新 SCM 和文件树，保持状态实时同步

## Fixes

- 修复内嵌在用户消息中的工具结果未同步任务状态的问题
- 修复用户中止会话后仍自动恢复的问题
- 修复 SCM 未跟踪文件显示问题，新增上下文压缩功能
- 修正会话 Git 变更统计
- 修复回退检查点识别与预览刷新
- 修复模型配置面板内容溢出被裁剪
- 保持终端面板挂载并设置默认工作目录

## Refactor

- 优化 MarkdownRenderer 渲染流程，避免 OOM（内存溢出）
- 调整全局任务看板的布局位置和间距
- 替换侧边栏静态 logo 为 SVG 图标
