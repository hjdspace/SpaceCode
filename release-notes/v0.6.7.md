## SpaceCode v0.6.7

本版本核心交付**多套模型配置切换（Profile）**功能，支持保存多组 API Key / Base URL / 模型配置并一键切换；同时完善开源文档体系，正式以 MIT 协议开源。

## Highlights

- **多模型配置 Profile**：支持保存多套 LLM 服务商配置（Base URL / API Key / 模型 / 上下文窗口），通过 ProfileCards 组件可视化切换，自动持久化到 `~/.spacecode/profiles.json`
- **Profile IPC 与 Store**：主进程新增 `profiles:load/save` IPC handler，preload 安全暴露，Pinia store 提供 7 个 actions 管理 profile 状态
- **自动迁移与容错**：首次启动自动将现有单一配置迁移为「默认」Profile；`profiles.json` 解析失败时自动备份损坏文件，避免配置丢失
- **静默切换 + Toast 提示**：切换 Profile 静默生效（仅影响新会话），并显示 Toast 通知，避免用户困惑
- **Markdown 本地图片解析**：新增本地图片相对路径解析能力，修复预览时图片加载 404 与 CSP 限制问题
- **正式开源**：新增 LICENSE（MIT）、CONTRIBUTING.md、SECURITY.md；README 补充路线图、贡献指南、致谢、社区与反馈、Star History 等开源必备章节
