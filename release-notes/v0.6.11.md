## SpaceCode v0.6.11

本版本为移动 App 带来完整的联网搜索能力（WebSearchProvider 抽象 + Jina/Tavily/Brave 三种 Provider）与基于 Termux 桥接的真 git clone 实现，同时新增会话管理国际化、上地项目管理、工具调用流式渲染、CodeViewer 搜索、面板性能优化与 OfficeCLI 应用内下载等多项改进。

## Highlights

- **手机端联网搜索**：实现 WebSearchProvider 抽象接口，新增 Jina（默认免 Key）/ Tavily / Brave 三种 Provider 与工厂模式，注册 web_search 与 fetch_url 工具到 Agent（归类为只读权限），MobileConfig 新增 searchProvider/searchApiKey 字段及持久化，设置页新增联网搜索配置卡片，i18n 文案支持中英双语
- **真 git clone（Termux 桥接）**：新增 GitCloneService 分四段执行 git clone，引入 TermuxReadiness 三态检测（未安装/未授权/就绪），GitPlugin 对 pull/push/fetch 注入临时 credential helper（token 不持久化），并配套设置页引导卡片与 clone 前置二选一弹窗（Termux 未就绪时引导安装或降级 zipball）
- **会话管理国际化**：实现会话列表页面的国际化与项目分组功能
- **上地项目管理**：聊天工作区新增上地项目管理功能
- **工具调用流式渲染**：实现工具调用流式渲染与状态展示，合并流式文本组件到 Markdown 渲染器，优化工具卡片空值展示逻辑
- **CodeViewer 搜索**：CodeViewer 新增搜索功能与快捷键（Ctrl+P 快速打开、Ctrl+F 查找）
- **面板性能优化**：优化分栏面板拖动性能与可调整大小逻辑
- **OfficeCLI 应用内下载**：实现 OfficeCLI 二进制应用内下载与进度跟踪，构建在缺失二进制时可继续

## Fixes

- 修复会话切换时的组件卸载断言错误
- 修复 MobileConfig.load 未读取搜索配置
- 修复 Android 语音输入无反应（切换至 Whisper API 方案）
- 修复 shell_plugin termuxReadiness 迁移回归
- 修复多处异步状态异常与测试适配问题
- 实现 rate limit 错误处理
