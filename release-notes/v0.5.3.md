## SpaceCode v0.5.3

本版本重点增强了 MCP 服务器管理能力，新增内置钩子系统与工作模式生态，并重构了弹窗与工具卡片系统。

## Highlights

- **MCP 服务器主动注入**：实现 MCP 服务器主动注入与活跃列表查询能力，提升 MCP 集成体验
- **内置 MCP 依赖安装**：新增内置 MCP 依赖一键安装与状态检测功能，简化 MCP 环境配置
- **内置 MCP 管理**：新增内置 MCP 服务器管理支持，统一管理内置与外部 MCP 服务
- **产物面板**：新增产物面板功能，支持查看和管理工作产出文件
- **Workspace 工具栏**：新增工作区工具栏，支持 HTML 产物自动预览与链接正则修复
- **工作模式与专业助手**：新增工作模式与专业助手生态，拓展应用场景
- **内置钩子系统**：新增内置钩子系统与聊天上下文工具栏，增强会话生命周期管理
- **原型页面**：新增三个原型页面，包含 Dashboard 配置页和终端卡片样式

## Fixes

- 修复 ReadToolCard 文件路径未解析的问题
- 修复 BashToolCard 颜色代码错误和注释翻译
- 改进 build 命令错误处理和消息提示
- 修正 PushPlus 通知的请求域名

## Refactor

- 重构 Bash 工具卡片，移除内嵌终端并新增终端页跳转
- 优化外部链接打开与产物扫描逻辑，新增路径解析工具
- 重构弹窗系统，统一使用组合式 API 弹窗管理
- 重构 ECC hook 实现，新增路径修复功能
- 全局化多语言支持，替换硬编码文本
