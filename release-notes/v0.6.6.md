## SpaceCode v0.6.6

本版本聚焦桌面宠物系统的完整实现，包含独立窗口与悬浮嵌入双模式、18 种内置宠物物种、LLM 代理深模块、状态管理与设置面板等全套功能；同时新增了聊天工具搜索与执行能力，并完成工具卡片重设计对齐。

## Highlights

- **桌面宠物系统**：完成完整实现，支持独立窗口与悬浮嵌入双模式运行
- **18 种内置宠物**：新增 18 种内置宠物物种和精灵 SVG 组件
- **宠物状态管理**：新增 petStore Pinia 状态管理与 API 聚合层
- **宠物 LLM 代理**：新增 petLLMProxy LLM 代理深模块，驱动宠物智能反应
- **独立窗口管理**：新增 petWindowManager 独立窗口管理及 IPC 集成
- **宠物设置面板**：新增 Card / PreviewHeader / Gallery / Creator / BehaviorConfig 等设置组件
- **宠物交互 composable**：新增 usePetDrag / usePetReaction composable
- **宠物反应气泡**：新增 PetReactionBubble 反应气泡组件
- **上下文事件反应**：新增上下文事件触发宠物反应机制
- **聊天工具增强**：新增搜索额外工具和执行额外工具功能
