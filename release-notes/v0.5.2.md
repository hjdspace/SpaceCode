## SpaceCode v0.5.2

本版本新增链路追踪（Trace）功能、会话上下文面板系统和 Git 事件驱动刷新，同时优化了子代理协作体验和打包体积。

## Highlights

- **链路追踪**：实现完整的 Trace 功能，支持会话列表、会话详情与调用详情展示，重构 TraceListPage 和 TraceSessionPage 组件界面
- **会话上下文面板**：新增会话上下文面板系统，支持面板展开模式，新增创建分支和 Git 图表面板
- **Git 事件驱动刷新**：实现 git 仓库变更事件驱动刷新替代轮询，启动监听前确保开启项目文件监听
- **Git 推送功能**：新增 Git 推送功能，优化分支获取逻辑与环境面板
- **子代理协作**：完善子代理（sidechain）协作与终端体验
- **fs:readFileAsBase64**：新增 Electron API 支持以 Base64 读取文件
- **前端代码审查技能**：新增前端代码审查技能及相关参考文档

## Fixes

- 修复子代理会话重复渲染与错误渲染问题
- 修复打开终端前未关闭各类全屏面板的问题
- 修复 trace 相关 IPC 调用的异步处理问题

## Refactor

- 优化环境面板适配逻辑，使用 ResizeObserver 动态调整布局
- 替换环境面板为玻璃态浮动面板并优化交互
- 优化实时子代理消息的归属与展示逻辑
- 抽离聊天输入框相关逻辑为独立组合式函数
- 替换 console 日志为 debug 日志
- 同步 stripSystemReminders 逻辑与引擎行为
- 重构会话配置与进程管理代码
