## SpaceCode v0.6.10

本版本为移动 App 带来完整的内置终端、Shell 命令执行、结构化 Git 工具、Python 沙盒与权限拦截器，重写聊天输入框组件群（附件、@mention、模型选择器），并新增 Agent 续跑能力（maxTurns 提升至 150，截断后可一键继续上次任务）。

## Highlights

- **内置终端与 Shell 执行**：实现内置终端与 Shell 命令执行能力，支持在工作区内执行命令；无工作区时回退到 homeDir 确保终端和 Shell 工具可用
- **GitPlugin**：实现 12 个结构化 Git 工具，覆盖常用 Git 操作
- **Python 沙盒与 PythonPlugin**：实现 Python 沙盒配置与 PythonPlugin，支持在移动端执行 Python 代码
- **权限拦截器**：新增 CommandClassifier 命令危险等级分类、PermissionInterceptorPlugin 异步询问流程，并集成到本地 Agent 与 UI
- **Android Chaquopy 集成**：Android Chaquopy 集成与 git 二进制构建脚本（注：因不兼容 Gradle 9 暂时禁用）
- **聊天输入框重写**：重写聊天输入框组件，新增模型选择器、@mention 选择浮层、附件选择底部抽屉、附件 Chip 组件
- **ChatState 扩展**：扩展 ChatState / ChatNotifier 支持附件与模型切换，新增 ModelService 与 modelsProvider，添加 saveModel 方法
- **CommandMenu**：使用 CommandMenu 替换 SkillCommandMenu 并支持命令分组
- **Agent 续跑能力**：maxTurns 由 8 提升至 150（参考 claude-code 安全闸 MAX_GOAL_TURNS=150），complete() 返回 AgentRunResult 透传 stopReason；maxTurns 截断时在输入框上方显示「继续上次任务」横幅，点击复用历史再次发起 Agent 调用
- **i18n 完善**：新增工作区相关多语言文案，补全聊天输入框 i18n 字典并新增 widget 测试

## Fixes

- 恢复 WorkspaceToolbar 至输入框下方
- 修复聊天输入栏工具按钮超出屏幕的问题
