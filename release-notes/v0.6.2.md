## SpaceCode v0.6.2

本版本完成了设计工作台的整体重构，新增设计模式聊天界面、模板与设计系统选择器、Browser Use 浏览器自动化、子代理工具卡片重设计等核心能力，并带来 MCP 服务管理、设计系统导出与多项交互体验优化。

## Highlights

- **设计工作台重构**：新增设计模式聊天界面 DesignChatPane、DesignComposer 输入框、文件工作区 DesignFileWorkspace，支持多 tab、预览源码切换与导出
- **模板与设计系统选择**：新增 TemplatePicker 模板选择器、WorkingDirectoryPicker 工作目录选择器，支持 inline 紧凑模式与官网链接展示
- **Browser Use 浏览器自动化**：新增 AI 操控网页浏览能力，支持 Cloud Browser、Agent 高级配置、MCP 浏览器自动化，以及镜像源加速安装与桌面 LLM 配置复用
- **子代理工具卡片重设计**：子代理卡片改为行内可展开行并支持流式输出，新增 Bash/Write/Edit/Read 专用工具卡
- **结构化设计卡片**：新增 OdCard 组件，支持 brand/direction/artifact/generic 四种类型展示
- **MCP 服务管理**：新增 MCP 服务管理、设计系统导出及体验优化
- **助手画廊**：重构助手画廊 UI，新增悬停详情与能力展示
