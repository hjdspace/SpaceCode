## SpaceCode v0.6.3

本版本重点新增 H5 移动端远程访问与镜像会话能力，并重构了设计会话的消息渲染与自动更新体验。

## Highlights

- **H5 跨端消息同步**：实现 H5 与桌面端消息同步，支持跨端访问。
- **H5 镜像会话**：完整实现 H5 移动端镜像会话功能。
- **H5 远程访问**：新增 H5 远程访问入口，支持手机浏览器连接桌面端。
- **RTK 下载与校验**：完善下载状态管理，新增 RTK 钩子安装校验与 Token 优化。
- **设计系统工作区**：支持在设计会话中切换工作目录，优化相关选择器与面板交互。
- **设计消息渲染**：新增 TurnSummaryBar 轮次摘要条与 DesignBlocks UI 块渲染，重构 MessageList/MessageItem 设计模式。
- **自动更新**：实现退出时静默安装更新、静默自动下载与重试逻辑，新增 AboutSettings 关于设置面板。
- **Agent 技能文档**：新增多个 Agent 技能文档，扩展工程技能体系。

## Fixes

- 确保在启动进程前目标工作目录存在。
- 简化 TitleBar 更新按钮逻辑与样式。
- 移除 UpdateNotification 组件及相关代码。

## Refactor

- 优化 RTK 下载与多平台支持逻辑。
- 重构 Agent 任务状态同步逻辑。
- 重构 WorkingDirectoryPicker、DesignChatPane、MessageList、MessageItem 等设计相关组件。
- 将轮次摘要重构为使用 TurnSummaryBar 组件。
- 移除 MessageItem 设计模式相关测试。
