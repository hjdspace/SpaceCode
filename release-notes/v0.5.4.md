## SpaceCode v0.5.4

本版本重点推出了分屏多会话布局系统与工作模式核心功能，全量接入 officecli 技能体系，并新增 Computer Use MCP 服务器与产物汇总卡片等能力。

## Highlights

- **分屏多会话布局**：实现中央区分屏多会话布局系统，支持拖拽调整面板、标签页独立管理与完整分屏功能，大幅提升多任务并行效率
- **工作模式核心功能**：完成工作模式核心功能开发，优化桌面程序办公模式与办公助手体验
- **officecli 技能体系**：全量接入 officecli 技能体系，新增多套 PPT 样式与自定义助手功能，为办公模式添加集成文档
- **产物汇总卡片**：新增 ArtifactSummaryCard 组件，支持按轮次收集和展示办公产物，并补充多语言文案
- **Computer Use MCP**：新增内置预打包 Computer Use MCP 服务器并完成迁移适配，为系统提示添加可用性说明
- **MCP 加载优化**：新增 Claude Code CLI 加载状态显示与安装进度优化，提升 MCP 环境配置体验
- **图片转 Base64**：新增图片转 base64 能力，优化会话和预览面板体验
- **自动更新增强**：新增下载准备状态与超时处理逻辑，提升更新可靠性
- **技能分类管理**：新增技能分类过滤、分类统计与分类覆盖映射，优化技能/技能包分类逻辑

## Fixes

- 修复单叶模式下关闭会话标签后内容不同步的问题
- 重置编辑器高度以修复输入框残留扩展状态
- 修正权限模式启动与恢复逻辑
- 修复 hexToRgba 不支持 3 位十六进制颜色的问题

- 优化终端面板体验与状态管理
- 重构分栏布局的 PaneHeader 与样式逻辑
- 优化 officecli 二进制文件处理逻辑
- 抽取共享的 iconFor/formatSize 工具函数
- 统一替换 emoji 头像为 lucide 图标并优化展示
- 调整产物面板显示逻辑
- 优化权限模式逻辑与会话历史管理
