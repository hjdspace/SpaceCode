## SpaceCode v0.5.1

本版本新增文件快速搜索面板和更新日志功能，修复打包后代理子进程加载问题，并统一了全局字体与样式规范。

## Highlights

- **文件快速搜索**：新增文件快速搜索面板与右侧面板启动器，快速定位项目文件
- **更新日志**：新增更新日志功能，支持本地/远程加载和弹窗展示，方便查看版本变更
- **Pi SDK 一键安装**：一键安装 Pi SDK 并优化引擎运行时，简化引擎配置流程
- **私有仓库更新**：为私有仓库更新添加 GitHub token 支持，支持私有仓库自动更新

## Fixes

- 修复快捷键类型定义并修复引擎选择逻辑
- 修复打包后代理子进程无法加载脚本的问题
- 优化 CI changelog 模板适配 commit 模式

## Refactor

- 统一全局字体配置，移除硬编码字体样式
- 移除动态导入改用静态导入 fs 和 path 模块
- 替换 array.at(-1) 为更兼容的写法
- 重构 Vite 配置并优化项目构建流程
- 用手动 release-notes 替换 changelog-builder
