BashToolCard 展开样式优化原型

核心改动：展开后默认用「终端窗口」样式统一渲染命令 + 输出，不再分成两个独立块。
命令以 $ 提示符形式显示在终端内，输出紧跟其后，视觉上接近真实终端。
下方展示折叠态、运行中、完成、错误四种状态。

1. 折叠状态（点击 header 展开）

Bash npm run build 2.3s
