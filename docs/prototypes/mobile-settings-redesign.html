SpaceCode Mobile — 设置页面 UI 优化

基于 impeccable/layout 空间设计原则，对当前手机端设置页进行布局分析与优化。以下展示 5 种方案，请在手机模拟器宽度（375px）下浏览。

当前布局问题诊断

  • 间距无系统：16/12/8/10/24/4/2 等任意值混用，10px 不属于 4pt 刻度，缺乏统一间距尺度 Spacing
  • 层级弱：标题仅 17px、分区头仅 12px + 0.5 透明度，眯眼测试无法区分主次 Hierarchy
  • 节奏单调：每个分区都是「头→列表」的重复结构，无紧凑/宽松交替，无视觉焦点 Rhythm
  • 卡片使用不一致：连接区单独用卡片，其余分区用裸 ListTile + border-bottom，容器策略混乱 Cards
  • 表单拥挤：引擎配置 3 个输入框间距仅 10px，缺少语义分组 Density
  • 主题选择器过小：色块高 48px，标签 11px，在 4 列中过于紧凑 Touch Target
  • 关于区空洞：仅 2 行文字，无视觉重量，与上方分区无过渡 Hierarchy
方案 A — 分组卡片
iOS Settings 风格：大标题 + 分组圆角卡片，间距系统 4/8/12/16/24/32，分区头使用大写字母 + 字间距强化层级。
