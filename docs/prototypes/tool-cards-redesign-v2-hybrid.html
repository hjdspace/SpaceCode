V2 设计要点：
1. 折叠态头部改为行内文本行，与 Subagent 风格一致，不破坏时间线阅读流。
2. 展开后仍使用圆角卡片容器承载内容，让终端、代码块、diff 有清晰的边界。
3. 操作按钮（在终端打开、在面板打开）放在头部右侧，保持可发现性。
4. 卡片容器使用主题色背景，与当前组件库的视觉层次保持一致。
5. 相比 V1 纯行内，V2 更适合包含大量代码/终端输出的工具卡片。
