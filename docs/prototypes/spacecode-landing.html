Computer Use 桌面控制

基于 cua-driver 的后台桌面操控，AI 可截图、点击、输入文字、滚动、拖拽，真实光标始终不动。

Browser Use 浏览器自动化

AI 操控真实浏览器完成网页浏览、信息检索、表单填写，支持本地与云端浏览器。

🤖

双引擎多模型架构

Claude Code CLI + Pi Engine 双引擎，支持 Anthropic、OpenAI、Gemini、Grok、DeepSeek 等主流 LLM。

🔌

MCP 协议生态

支持 stdio / SSE / HTTP 三种传输方式，内置 Computer Use 与 Browser Use MCP 预设，可视化配置。

技能库与智能体系统

50+ 内置技能（前端设计、PPT 生成、代码审查、TDD 等），70+ 内置 Agent，支持工作流编辑。
