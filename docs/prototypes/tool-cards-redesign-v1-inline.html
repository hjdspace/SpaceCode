V1 设计要点：
1. 所有工具卡片统一为行内文本行，无卡片背景/边框，与 Subagent 卡片风格一致。
2. 折叠态显示：图标 + 工具名 + 中点分隔 + 目标内容（命令/路径/查询/URL）+ 可选元信息。
3. 展开后使用带标题的浅色区块承载终端、代码、diff、搜索结果等富内容。
4. 运行/完成/错误状态仅通过图标颜色表达，不破坏时间线文本流。
5. 保留了 Bash 终端窗口、Read 代码高亮、Edit diff 等专业内容容器。
