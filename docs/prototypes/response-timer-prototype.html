LLM 用时计数原型 Prototype
用时计数功能说明
本原型展示了三种状态下的用时计时器表现：
① 运行中（Running）：时间轴头部显示实时递增计时器，带脉冲动画，与状态徽章并排展示；
② 已完成（Completed）：计时器停止，显示最终用时，样式变为低调灰色；
③ 错误状态（Error）：计时器停止，显示用时并标红；
每轮对话结束后，底部还会展示 用时汇总条（总时长 + 模型 + tokens）。

SpaceCode 完成

思考 1.8s ▾

ref 和 reactive 是 Vue 3 响应式系统的两个核心 API，主要区别如下：

1. 适用类型
• ref：适用于所有类型（基本类型 + 对象），通过 .value 访问
• reactive：仅适用于对象/数组，直接访问属性

2. 推荐用法
• 简单值（string, number, boolean）→ 用 ref
• 表单对象、配置对象 → 用 reactive

什么是 debounce？
SpaceCode 完成
0.8s

Debounce（防抖）是一种编程技术，用于限制一个函数在特定时间内只能执行一次。如果在等待期间再次触发，则重新计时。

典型应用：搜索框输入、窗口 resize 事件、按钮防连点。
