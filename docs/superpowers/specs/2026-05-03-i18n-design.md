# 中英文切换功能设计文档

## 目标

为 SpaceCode 桌面应用添加国际化（i18n）支持，允许用户在设置中切换中文和英文界面，默认语言为中文。

## 架构

采用 **vue-i18n** 作为国际化框架，语言包按功能模块组织（settings、chat、common 等）。语言设置存储在 Pinia store 中并持久化到 localStorage，应用启动时读取并设置当前语言。

## 技术栈

- **vue-i18n@9** - Vue 官方国际化插件
- **Pinia** - 状态管理，存储语言设置
- **localStorage** - 持久化语言偏好

## 文件结构

```
src/
├── i18n/
│   ├── index.ts              # i18n 初始化配置
│   ├── locales/
│   │   ├── zh-CN.ts          # 中文语言包
│   │   └── en-US.ts          # 英文语言包
│   └── types.ts              # 类型定义（可选）
├── stores/
│   └── settings.ts           # 添加 language 字段
├── components/settings/
│   └── GeneralSettings.vue   # 添加语言选择器
└── main.ts                   # 引入 i18n
```

## 语言设置位置

**位于 General Settings（通用设置）页面顶部**，原因：
1. 语言是应用级基础配置，与登录方式同属核心设置
2. 符合用户习惯（VS Code、Chrome 等主流软件均将语言放在通用设置）
3. 实现简洁，无需新增标签页

## 界面改动

### GeneralSettings.vue

在页面顶部添加语言选择器：

```vue
<!-- Language Selection -->
<div class="form-group">
  <label class="form-label">{{ $t('settings.language') }}</label>
  <div class="language-options">
    <button
      v-for="lang in languages"
      :key="lang.id"
      class="language-card"
      :class="{ active: currentLanguage === lang.id }"
      @click="selectLanguage(lang.id)"
    >
      <span class="language-flag">{{ lang.flag }}</span>
      <span class="language-name">{{ lang.name }}</span>
      <Check v-if="currentLanguage === lang.id" :size="16" />
    </button>
  </div>
  <span class="form-hint">{{ $t('settings.languageDesc') }}</span>
</div>

<div class="divider"></div>
```

### 样式

语言选择器采用与主题选择器相同的卡片式布局，保持一致性：
- 两个选项卡片横向排列
- 选中状态显示勾选图标
- 包含国旗/语言标识

## 数据流

```
用户选择语言
    ↓
GeneralSettings.vue 调用 store 方法
    ↓
settings.ts 更新 language 状态
    ↓
保存到 localStorage
    ↓
调用 i18n.global.locale.value = newLocale
    ↓
界面即时切换语言
```

## 默认语言

- **默认语言：中文 (zh-CN)**
- 首次启动时检测系统语言，如果是中文则使用中文，否则使用英文
- 用户可手动切换，偏好持久化存储

## 边界情况处理

1. **首次启动**：检测 `navigator.language`，如果是 `zh` 开头则默认中文
2. **语言包缺失**：fallback 到英文
3. **动态切换**：切换后立即生效，无需重启应用
4. **未翻译文本**：显示 key 名，便于开发时识别

## 测试要点

1. 语言切换后所有界面元素正确更新
2. 刷新页面后语言偏好保持
3. 默认语言逻辑正确
4. 语言选择器 UI 正常显示和交互
