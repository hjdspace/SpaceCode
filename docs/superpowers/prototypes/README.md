# 设计模式重构 HTML 原型

`design-mode-refactor-prototype.html` 是设计模式重构的高保真 HTML 原型，用于在实现前可视化验证设计方向。

包含：
- 3 种状态（空态 / 活跃会话 / 重试）
- 主题切换（深色 / 浅色）
- 全部 block 类型演示（text / thinking / tool-group / od-card / question-form / next-steps / status）
- 可拖拽分屏、tab 切换、视图模式切换、Toolbox 面板等交互元素
- 模拟 SpaceCode 的 CSS 变量 token 体系
