编码模式

面向开发者的 AI 编程助手，深度集成 Claude Code 引擎，提供代码编写、调试、重构到项目管理的全流程辅助。

  • AI 对话编程 — 自然语言描述需求，自动读写文件、执行命令
  • 工具调用卡片 — 13+ 工具的可视化卡片，每步操作清晰可见
  • 代码 Diff 预览 — 基于 @git-diff-view 的行级差异对比
  • 集成终端 — xterm.js + node-pty，多标签页无缝协作
  • Git SCM 面板 — 暂存、提交、推送、分支管理一应俱全
  • 对话回滚 — 4 种回滚策略覆盖各类场景
  • 权限控制 — 自动/手动/拒绝/建议四种模式
💼

办公模式

面向职场场景的 AI 办公助手，内置 20+ 专业助手，直接生成可编辑的 .pptx、.docx、.xlsx 文件。

  • 专业办公助手 — PPT 制作、Word 文档、Excel 表格等 20+ 助手
  • 真实文件产出 — 直接生成可编辑的 Office 文件
  • 产物汇总卡片 — 自动收集展示生成的文件
  • 文档实时预览 — HTML 渲染、截图缩略图、实时观看
  • 自定义助手 — 配置系统提示词、技能绑定与推荐 Prompt
  • officecli 技能体系 — PPT 多套样式、Word 表单等
  • 工作空间管理 — 产物统一存放在 outputs/ 目录
