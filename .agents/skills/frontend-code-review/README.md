# 前端代码审查技能

专为前端项目打造的代码审查工具，支持React、Vue、Next.js、Nuxt.js、TypeScript、Tailwind等主流前端技术栈。

## 核心能力
- 全维度代码审查：功能正确性、性能、安全、可维护性、用户体验、最佳实践
- 专项检查支持：React/Vue/TypeScript专项问题识别
- 分级问题输出：按严重程度分类，优先解决高风险问题
- 可落地的修复建议：每个问题都提供具体的修复方案

## 适用场景
- Pull Request代码审查
- 项目上线前代码质量检查
- 前端技术栈升级前的代码评估
- 前端团队代码规范落地检查
- 新人代码质量指导

## 使用方式
将需要审查的前端代码文件或代码片段提供给技能，即可得到结构化的审查报告。
